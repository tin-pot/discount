Markdown tool “discount” on Windows

This repository is a clone of David Parsons' implementation in C of John Gruber's Markdown markup language, with added support to build the package with Visual Studio on Windows.

Modifications

  • Added pre-configured headers and sources, so that no configure.sh run is needed.

  • Added project files for all executables and libraries.

  • Added a project getopt(with sources, of course) containing a MSVC adaptation of GNU getopt. NOTE You must link to GNU getopt as a separate DLL, otherwise you would violate the license.

  • Added a project freegetopt containing a free BSD- licensed implementation of getopt. This may be linked statically and is thus used by default.

Building/Installing

This should build “out of the box”: Open the Visual Studio “solution file” in the directory VS2003, and build. (The project files are for VS .NET 2003, but will get upgraded automatically when imported into a newer version of Visual Studio.)

To install, copy the binaries (in the Win32/Release directories) into a directory which is in your PATH. Don’t forget to put getopt.dll beneath them.

discount on Windows

The repo tin-pot/discount is a clone of David Parsons' implementation in C of John Gruber’s Markdown markup language, with added support to build the package with Visual Studio on Windows.

The page you see is generated by mkd2html on Windows from Markdown text.

Modifications

  • Added pre-configured headers and sources, so that no configure.sh run is needed.

  • Added project files for all executables and libraries.

  • Added a project (with sources, of course) containing a MSVC adaptation of GNU getopt.

Building/Installing

This should build “out of the box”: Open the Visual Studio “solution file” in the directory VS2003, and build. (The project files are for VS .NET 2003, but will get upgraded automatically when imported into a newer version of Visual Studio.)

To install, copy the binaries (in the Win32/Release directories) into a directory which is in your PATH. Don’t forget to put getopt.dll beneath them.

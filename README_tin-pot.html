The Markdown tool “discount” on Windows:
“tin-pot/discount”

This repository is a fork of David Parsons' repository containing his implementation in C of (a processor for) John Gruber’s “Markdown” markup language, with

  • added support to build the package with Visual C++ on Windows,

  • and some minor new features.

See also the home page of this distribution at github: tin- pot/discount.

In the following, the term “UNIX” refers to all the UNIX-like platforms that discount does originally target.

1  Modifications

The modifications added in this distribution are:

  • Added pre-configured files

    • version.c,
    • mkdio.h, and
    • config.h,

    in the directory MSVC/ so that no configure.sh run—which generates these files on UNIX—is needed on Windows.

  • Added Visual Studio project files for all executables and libraries in MSVC/.

  • Added supporting files for building on Windows in MSVC/posc/, including the freegetopt implementation of UNIX’s getopt() function which is used by main.c.

  • Some new features are also available, see the CHANGES file for more details.

2  Building/Installing

The package should build “out of the box” with Visual Studio 2003.NET or Visual Studio 2008 or newer (newer versions of Visual Studio will import and upgrade the project files).

The generated executable are statically linked: they only depend on kernel32.dll. Place them wherever you like on your PATH.

3  About the Port

Porting to Windows was done using a fairly general technique, which presents to the compiler an environment of “faked” standard header files on top of the “real” header files of Visual C++. All this is contained in the MSVC/posc directory. See the MSVC/posc/README file there for explanations.

All discount source files in the project root directory are unmodified (except for new features, that is). Their dependency on functions like strncasecmp() (UNIX), or vsnprintf() (C99), or getopt() gets satisfied completely out of files in posc.

The included getopt() implementation is freegetopt, its license is in the file LICENSE-freegetopt.
